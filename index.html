📘 About FeelFusion

  • What is FeelFusion?
    A game hub powered by real-time emotion detection, adapting gameplay based on your feelings. It blends web tech, AI, and interactivity into a unified experience.
  • How Emotion Detection Works:
    Using face-api.js, our platform reads expressions from your webcam to adjust game difficulty in real time. Play harder when you're smiling — or take it easy when you're stressed.
  • Technologies Used:
    HTML5, CSS3, JavaScript, face-api.js (TensorFlow backend), Google Fonts, Visual Studio Code.
